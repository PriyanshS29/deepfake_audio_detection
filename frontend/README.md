# Deepfake Audio Detection — Frontend

This is a minimal Vite + React scaffold for the project.

Quick start (Windows Powershell):

```powershell
cd frontend
npm install
npm run dev
```

Open the dev server URL shown in the terminal (usually http://localhost:5173).

To connect to the backend, update API calls to the backend server URL (e.g., `http://localhost:5000`).
